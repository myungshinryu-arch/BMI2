1. 통합 포털 홈 사용법

통합 포털 홈은 대시보드 상태 모니터링, 네이버 스타일의 실시간 검색, 4대 핵심 모듈 딥링크 및 동기화 연혁 제어를 지원하는 관제 관문입니다.

메인 화면 핵심 구성 메뉴 및 세부 조작법

1

네이버 스타일 'INTEGRATED BM SEARCH' 통합 검색 바 제어

• 조작 대상: 화면 중앙에 배치된 대형 통합 검색창 (`#integrated-search`)
• 마우스 및 키보드 조작법:
1. 검색창 내부를 마우스로 클릭하여 타이핑 모드를 활성화합니다.
2. 찾고자 하는 브랜드명(예: Michelin), 타이어 코드, 컴파운드 물성 또는 전략 키워드를 1글자 이상 기입합니다.
3. 타이핑하는 즉시 검색바 하단에 타이어 BM, 컴파운드 BM, 시장/전략 범주별 검색 매칭 목록이 드롭다운으로 실시간 렌더링됩니다.
4. 매치된 목록 중 원하는 타깃 아이템을 클릭하면 관련된 개별 하위 시스템 분석 페이지로 즉시 자동 이동 및 포커싱됩니다.
5. 검색창 우측 끝에 생겨난 [X] 버튼을 누르면 입력한 텍스트가 모두 제거되며 검색 드롭다운이 닫힙니다.

2

상단 '통합 Report' 및 'AI에게 질문하기' 신속 연동 버튼 제어

• 조작 대상: 포털 헤더 우측 상단에 번쩍이는 숨소리 애니메이션이 이식된 2개의 특화 버튼
• 클릭 조작과 연동:
- [통합 Report 작성] 클릭 시: 1:1 브랜드별 제품 시트를 설계하고 브라우저 프린터 드라이버와 실시간 결합하는 다크 테마 레포트 생성기로 즉시 화면이 교체됩니다.
- [AI에게 질문하기] 클릭 시: 포털 데이터베이스 수치들을 실시간 로딩하여 자연어 문장으로 답변을 추출하는 전용 AI 대화방(`chat-bot`) 페이지로 다이렉트 전환됩니다.

3

수평 'ACTIVE MODULES BAND' 딥링크 버튼 제어

• 조작 대상: 검색 바 하단에 가로로 병렬 배열된 5대 핵심 모듈 카드 (시장/상품전략, Tire BM, Comp'd BM, 경쟁사 BI News, AI Simulator)
• 마우스 클릭 조작법: 각 카드 내부의 세부 항목 링크(예: [전체 뉴스 피드], [제조사 트렌드 분석], [컴파운드 탐색기], [대응 전략 & 로드맵] 등)를 마우스로 클릭합니다. 클릭 즉시 해당 분석 페이지가 열릴 뿐만 아니라, 해당 하위메뉴(탭)가 활성화된 완벽한 상태로 페이지 내에 정밀 딥링크 연동 점프가 수행됩니다.

4

Global Top4 비교 대시보드 및 'History' 연혁 팝업창 제어

• 조작 대상: 포털 홈 하단의 글로벌 매출/판매 동향 및 우측 상단의 [History] 링크
• 대시보드 조작법: [분석 연도] 및 [대상 세그먼트] 드롭다운 셀렉터를 토글하여 차트의 기하학적 수치 변동을 실시간 관제합니다.
• History 모달 조작법: 우측 상단의 [History] 텍스트 링크를 마우스로 클릭하면 화면이 어두워지며 팝업 모달창이 활성화됩니다. 스크롤하여 연도별 동기화 히스토리를 정독한 뒤, 모달창 우측 위 [X] 버튼을 클릭하거나 바깥 영역을 선택하여 모달창을 닫습니다.

2. 시장/상품전략 조작법

글로벌 거시 정보, 경쟁사 기술 경쟁력 오버레이, 대표 3대 상품의 세대별 성능 변천사, 중장기 실행 로드맵 타임라인을 확인하는 조작 가이드입니다.

내부 5대 핵심 하위메뉴(탭)별 정밀 화면 조작 단계

1

[Market & Sales Overview] 탭 및 권역 버튼 조작

• 실제 화면 위치: 시장전략 페이지의 첫 번째 서브 탭
• 조작 프로시저:
1. 화면 내에서 [Market & Sales Overview] 탭을 클릭하여 엽니다.
2. 하위 영역에 노출되는 [북미 시장] 및 [유럽 시장] 원형 토글 버튼을 교차 클릭합니다.
3. 선택된 권역에 따라 세그먼트별 점유율 비중 막대그래프와 분기별 판매량 추이 곡선이 즉각 재드로잉되는 것을 확인합니다.

2

[기술 경쟁력 Overview] 탭 레이더 차트 및 툴팁 조작

• 실제 화면 위치: 시장전략 페이지의 두 번째 서브 탭
• 조작 프로시저:
1. [기술 경쟁력 Overview] 탭을 마우스로 선택합니다.
2. 화면 중앙에 출력되는 거대한 성능 6대축 레이더 차트의 상단 범례 라벨(예: Michelin)을 클릭하여 오버레이 그래프를 숨기거나 다시 켭니다.
3. 마우스를 꼭짓점 좌표 근처로 호버(올려두기)하여 자사와 비교 메이커 간의 정밀 타깃 평점 지수(예: Hankook Wet: 9.2 vs Michelin Wet: 9.5) 편차 수치를 툴팁으로 추적 판독합니다.

3

[대표 상품 세대별 성능] 탭 변천사 분석 뷰 조작

• 실제 화면 위치: 시장전략 페이지의 세 번째 서브 탭
• 조작 프로시저:
1. [대표 상품 세대별 성능] 탭을 클릭하여 엽니다.
2. 메이커 및 대표 타이어 제품의 세대교체(예: 1세대 -> 2세대 -> 3세대)에 따르는 점탄성 지수($\tan\delta$) 및 마모 효율성 발달 추이를 테이블 및 막대 차트 상에서 수평 대조 탐색합니다.

4

[경쟁사 R&D 전략] 및 [대응 전략 & 로드맵] 탭 타임라인 제어

• 실제 화면 위치: 시장전략 페이지의 네 번째 및 다섯 번째 서브 탭
• 조작 프로시저:
1. [대응 전략 & 로드맵] 탭을 선택하여 당사의 2026~2030 중장기 마케팅 로드맵을 수동 스크롤바를 이동시켜 시간 순서대로 추적합니다.
2. 화면 상단의 [단기 보강 과제], [중기 개발 과제], [장기 선행 과제] 등 오렌지/보라색으로 강조된 컬러 뱃지 영역을 클릭하면, 하단 테이블에서 해당하는 상세 기술 실행 안으로 화면 초점이 정확히 이동하여 정독할 수 있게 됩니다.

3. Tire BM 분석기 조작법

완성 타이어의 그루브/패턴 실측 데이터, 제품의 라이프사이클 타임 맵, EV 특화 타이어 필터 정보 및 사내 분석 PDF 보고서를 다운로드하는 조작 안내서입니다.

내부 4대 핵심 하위메뉴(탭) 및 공통 필터링 조작 프로시저

1

[종합 분석 대시보드] 탭 대형 테이블 다차원 정렬 조작

• 실제 화면 위치: Tire BM 페이지의 첫 번째 서브 탭
• 조작법:
1. [종합 분석 대시보드] 탭을 선택합니다.
2. 하단의 수십 여 타이어 제품이 전개된 대형 격자 테이블에서 특정 열 헤더 제목(예: [Groove Depth] 또는 [Pitch Count], [UTQG])을 마우스 좌클릭합니다.
3. 수치가 가장 높은 순(내림차순)으로 전체 데이터가 재정렬되며, 한 번 더 헤더를 누르면 수치가 가장 낮은 순(오름차순)으로 테이블이 반전 정렬됩니다. 마우스 휠을 굴려 자사 수치와 경쟁사 실측 수치 간의 차이를 비교합니다.

2

[PLC Timeline Map] 탭 출시 연혁 및 가로 드래그 조작

• 실제 화면 위치: Tire BM 페이지의 두 번째 서브 탭
• 조작법:
1. [PLC Timeline Map] 탭을 누릅니다.
2. 화면 중앙에 나타난 브랜드별 타이어 모델 타임라인 맵 영역을 마우스로 클릭한 채 좌우로 드래그하거나, 마우스 휠을 굴려 2021년부터 2028년 이후의 출시-단종 궤적 블록들을 연도 순으로 역동적으로 추적 탐색합니다.

3

[EV 친환경 특화 보드] 탭 및 친환경 지표 필터링 조작

• 실제 화면 위치: Tire BM 페이지의 세 번째 서브 탭
• 조작법:
1. [EV 친환경 특화 보드] 탭을 활성화합니다.
2. 고하중 지수, 전기차 전용 저소음 흡음재 적용 유무, 회전저항 1~2등급 최적화 모델만 선별적으로 남길 수 있는 특화 필터 버튼들을 클릭 제어하여, 자사의 대표적인 iON 타이어와 경쟁 모델 간의 강성을 조작 비교합니다.

4

[보고서 라이브러리] 탭 PDF 리포트 파일 1클릭 다운로드 조작

• 실제 화면 위치: Tire BM 페이지의 네 번째 서브 탭
• 조작법:
1. [보고서 라이브러리] 탭을 엽니다.
2. 보관된 기존 사내 완성 타이어 벤치마킹 분석 PDF 내역 리스트 중에서 필요한 항목 행을 마우스로 클릭하여 선택합니다.
3. 우측에 생성된 [다운로드] 또는 [조회] 아이콘 버튼을 마우스로 클릭하면 로컬 저장소로 원본 PDF 문서가 전송되어 영구 저장됩니다.

★

[공통 기능] 사이드바 3대 필터 버튼 및 상단 형광강조 검색 제어

• 좌측 사이드바 필터 조작: [북미/유럽 권역], [HP/UHP/EV 세그먼트], [Summer/All-Season 계절] 칩 버튼을 마우스로 클릭/토글하여 우측 목록 데이터를 즉각 커스텀 축소합니다. 필터를 전체 해제하려면 상단의 [필터 초기화] 버튼을 누릅니다.
• 상단 실시간 검색창 조작: 상단 우측 검색 상자에 찾고자 하는 타이어 제품 키워드(예: Sport)를 타이핑하면, 엔터 키를 누를 필요 없이 입력과 동시에 우측 테이블에서 매칭되는 행전체가 오렌지색으로 실시간 형광 하이라이트 글로우 처리됩니다. [X] 버튼 클릭 시 강조가 즉각 소거됩니다.

4. Comp'd BM 분석기 조작법

ARES 점탄성 측정 그래프의 범례 동적 필터링, 정량 성분 엑셀 시트 뷰, 드롭다운 기반 분석 보고서 실시간 인포그래픽 렌더링, Magic Triangle 대조 요령 안내서입니다.

내부 4대 핵심 하위메뉴(탭)별 화면 마우스 조작법

1

[제품 성능 비교] 탭 9대 메이커 개별 독립 카드 & 동적 대시보드 필터 가이드

• 실제 화면 위치: Comp'd BM 페이지의 첫 번째 서브 탭
• 조작법 및 주요 개편 사양 (Premium v1.5 적용):
1. [제품 성능 비교] 탭을 엽니다. (기존 '종합 분석 대시보드'에서 폰트 크기 및 시인성이 대폭 강화되며 명칭이 세련되게 개편되었습니다.)
2. 상단 Season 및 Segment 동적 대시보드 필터창 활용:
  : 대시보드 상단에 새롭게 도입된 [Season(시즌)] 및 [Segment / 차종] 필터를 통해 원하는 분류(예: Summer, EV 등)를 선택합니다.
  : 필터를 적용하면 하단의 9대 메이커 카드 내 드롭다운 리스트에 해당 시즌 및 차종 조건에 부합하는 타이어 패턴들만 실시간으로 동적 정제되어 표출됩니다.
  : 필터를 전체 초기화하여 원시 데이터셋으로 돌아가려면 필터 패널 우측의 [초기화] 단추를 클릭합니다.
3. 9대 제조사 개별 독립 프리미엄 카드 레이아웃:
  : 기존에 하나의 거대한 카드 컨테이너 내부에 묶여 있던 제조사 카드들을 완전히 분리된 개별 독립형 대형 카드 섹션으로 전면 개편하였습니다.
  : 금호타이어(KUMHO)를 포함한 9대 글로벌 메이커 카드가 독립적으로 확장 정렬되어 가독성이 극대화되었습니다.
4. 시인성 및 가독성 개선:
  : 카드 내부의 모든 텍스트, 물리적 배합 지표 수치, 드롭다운 상자의 크기가 일제히 상향되었습니다.
  : 불필요했던 'Select Pattern' 라벨을 완전히 생략하고, 드롭다운 폰트 크기를 1.35rem 수준으로 대폭 키워 선택된 패턴명이 시원하게 한눈에 들어옵니다.

6. [배경 그래픽] 체크박스에 마우스로 정확히 체크 기입하여 활성화합니다. (배경 그래픽이 꺼져있을 경우, 다크 테마 카드 프레임 그라데이션 및 차트 눈금선이 모두 백지로 공백 인쇄되는 심각한 시인성 누락이 발생합니다.)
7. 미리보기에서 정상적인 다크 테마 인포그래픽을 최종 확인한 뒤, 우측 하단 '인쇄' 또는 '저장' 버튼을 클릭해 영구 PDF 파일로 저장합니다.

8. AI에게 질문하기 사용법

포털 내 원천 데이터베이스 수치들 및 실시간 크롤링된 구글 뉴스 데이터와 결합하여, 자연어 질문창 입력을 통해 타이어 거동, 분석 기법 추천, 경쟁사 최신 R&D 위협 요소를 실시간으로 질문하고 답변을 주고 받는 조작 안내서입니다.

대화형 AI 가이드 인풋 전송 및 활용법

1

대화형 채팅 인풋창 클릭 및 뉴스 RAG 질의 기입

• 조작 대상: 화면 최우측 상단 [AI에게 질문하기]를 클릭해 진입한 챗봇 대화방 하부의 인풋창 (`#chat-input`)
• 조작법: 챗방 하부의 텍스트 입력 칸을 마우스 좌클릭한 후, 키보드로 벤치마킹 지표 외에 최근 업데이트된 크롤링 뉴스 기반의 RAG 질의(예: "미쉐린의 최근 R&D 기술 관련 뉴스 요약해줘" 또는 "콘티넨탈의 ESG 친환경 타이어 특허 동향 알려줘")를 자연어로 작성 기입합니다.
• 작동 및 Fallback 프로세스: AI 챗봇이 사내 데이터베이스 수치 외에, 실시간 크롤링 및 가공 완료된 BI News 원천 텍스트 데이터를 검색 매핑(RAG)하여 입체적으로 대조 요약합니다. 만약 인터넷 지연이나 데이터 누락 등으로 최신 기사 매칭에 실패하는 경우, "사내에 축적된 타이어 고무 배합 및 물성 벤치마킹 표준 DB 기준"으로의 자동 우회(Fallback) 답변 프로세스가 가동되므로 질의의 연속성이 완전 보장됩니다.

2

[전송] 버튼 클릭 또는 키보드 Enter 입력 실행

• 조작 대상: 인풋창 우측에 장착된 비행기 날개 모양의 [전송] 아이콘 또는 키보드 엔터키
• 조작법: 문장 기입을 마친 후 마우스로 전송 아이콘을 클릭하거나 키보드의 Enter 키를 누르면 텍스트가 위쪽 대화형 타임 버블 영역으로 즉시 페이드인 되며 전송 완료됩니다.

3

AI 실시간 추천 분석 피드백 버블 수집 및 정독

• 조작 대상: 대화창 영역에 순차적으로 로딩 실현되는 오렌지/다크 그레이 컬러 말풍선 버블
• 활용 요령: 인공지능이 포털 데이터베이스 실측값 및 BI News 데이터를 동역학적으로 참조 파싱하여 작성한 "추천 실리카Phr 범위 및 경쟁사 위협 기술 트렌드" 답변을 정독하여, AI 시뮬레이터 배합 설계 및 컴파운드 벤치마킹 시 타당성 근거로 폭넓게 활용하십시오.

9. FAQ & 화면 트러블슈팅

플랫폼 내 필터, 검색 강조 기능 및 보고서 출력 기능 조작 중 예기치 않게 발생하는 화면 이상 현상에 대한 즉각 대처법 모음입니다.

Q1

Q. 검색 바에 단어를 입력했는데 하단 테이블에 오렌지색 하이라이트 행이 하나도 나타나지 않습니다.

• 원인: 입력한 검색어 키워드가 테이블 내 셀 내용과 완벽히 불일치하거나, 영어 대소문자 매칭 오차 혹은 앞쪽 사이드바에서 지정한 '글로벌 필터(권역/세그먼트)' 조건에 의해 타깃 데이터 행 자체가 목록에서 가려져 숨김 상태인 것이 원인입니다.
• 해결 조치 단계:
1. 좌측 사이드바 필터 패널 영역에서 최상단의 [필터 초기화] 버튼을 누릅니다.
2. 검색창 내에 오타가 없는지 검증하시거나 검색창 우측 끝의 [X] 지우기 버튼을 눌러 텍스트를 깨끗이 지웁니다.
3. 다시 원시 상태에서 비교하고자 하는 제품 코드의 일부 단어(예: Michelin 또는 S1)를 정밀 기입하면 정상 강조가 재기동됩니다.

Q2

Q. AI 시뮬레이터에서 실리카/커플링제 슬라이더 노브를 마우스로 조작했는데 가끔 예측 차트의 선이 반응하지 않고 멈춰 있습니다.

• 원인: 브라우저 내부의 렌더링 스레드가 일시적으로 과부하 상태에 빠졌거나, 마우스 조작 속도가 초당 60프레임을 넘어서며 연산 버퍼 루프에 정체가 발생했기 때문입니다.
• 해결 조치 단계:
1. 슬라이더 노브를 한 번 클릭해 활성화시킵니다.
2. 마우스 드래그를 멈추고, 키보드의 좌/우 방향키(←, ➡️)를 한 번씩 누릅니다.
3. 키보드 이벤트를 통해 AI 모델 연산 매핑 함수가 강제로 다이렉트 재호출되며 정지되었던 차트 선이 새로운 값으로 부드럽게 다시 갱신 렌더링됩니다.

Q3

Q. 통합 Report 인쇄 버튼을 클릭했는데 인쇄 팝업 창 안에서 카드 배경 색상들과 동적 라인 그래프 그림이 하얗게 누락되어 보입니다.

• 원인: 사용하시는 크롬/엣지 웹 브라우저의 기본 인쇄 템플릿 사양에서 성능과 토너 절약을 위해 '배경 요소를 소거'하도록 기본 설정되어 있기 때문입니다.
• 해결 조치 단계:
1. 인쇄 미리보기 설정창(Ctrl+P 호출창)의 우측 제어 서브메뉴 영역을 하단으로 스크롤하여 '설정 더보기' 또는 '옵션' 아코디언 메뉴를 엽니다.
2. 옵션 항목 목록 중 [배경 그래픽] 체크박스에 반드시 마우스로 체크 표시를 활성화합니다.
3. 체크가 켜지는 순간 미리보기창 내에 감각적인 다크 테마 카드 배경 및 그라데이션 차트 라인 전체가 실물 화면과 100% 동일하게 복원되어 선명하게 표출되는 것을 확인한 뒤 인쇄/PDF 저장을 실행합니다.

Q4

Q. 하위 개별 시스템으로 이동한 직후에 마우스를 오버하지 않았는데도 왼쪽 사이드바 상태창이 저절로 튀어나와 대시보드를 가립니다. 혹은 마우스를 밖으로 뺐는데도 상태창이 자동으로 들어가지 않습니다.

• 원인: 이전 페이지에서 사이드바 내부 링크를 클릭할 때 마우스 커서가 사이드바 영역 내에 머물러 있어 '강제 오픈 상태'로 브라우저 메모리에 임시 기록되었거나, 마우스 이탈 감지 오동작이 일어났기 때문입니다.
• 해결 조치 단계 및 개선 스펙 (Premium v1.5 적용):
1. **자동 숨김 전면 탑재**: 모든 개별 시스템(Tire BM, Comp'd BM, 시장/전략, 경쟁사 BI News)의 사이드바 디자인이 290px 너비, Outfit 폰트, 1.05rem 글자 크기, 700 굵기로 일치화되었습니다.
2. **마우스 이탈 감지 즉시 숨김**: 마우스 커서를 왼쪽 사이드바 영역 바깥으로 빼내면 mouseleave 이벤트가 즉각 동작하여 상태 바가 미끄러지듯 스르르 닫힙니다.
3. **자동 오픈 메모리 버그 해결**: 페이지 이동 시 사이드바가 저절로 확 펼쳐진 채 고정되던 메모리 유지 현상을 원천적으로 제거하였으므로, 사용자가 마우스를 다시 오버하기 전까지는 깨끗한 대시보드를 넓게 관제하실 수 있습니다.

Q5

Q. 제품 성능 비교 카드 내의 글씨가 대폭 커졌는데, 화면 해상도나 레이아웃에 맞춰 스크롤 없이 한눈에 볼 수 있는 방법이 있나요?

• 원인 및 사양 개선 내용 (Premium v1.5 적용):
1. **연구원 가독성 피드백 반영**: 카드 내 제조사 브랜드명을 1.95rem으로, 드롭다운은 1.35rem으로 확대하고 배합 평균 및 성능 지수 텍스트 크기를 전반적으로 상향하여 시인성을 극대화하였습니다.
2. **드롭다운 시각 직관성 확보**: 불필요한 'Select Pattern' 라벨을 완전히 생략하고 선택된 패턴명과 샘플 개수(N)가 넓은 영역에서 선명하게 표시되도록 정리했습니다.
3. **최적 권장 렌더링 비율**: 글씨 크기 증가로 인해 레이아웃에 줄바꿈이 빈번히 발생하거나 화면 높이가 모자라 보일 경우, 브라우저 축소 비율(Ctrl + 마우스 휠 아래 스크롤)을 80%~90% 정도로 세팅하시면 초고해상도의 9대 메이커 비교 장표를 한눈에 관제하실 수 있습니다.

Q6

Q. Compound BM 제품 성능 비교 페이지에서 특정 제조사의 패턴 드롭다운을 열었는데 원하는 패턴이 보이지 않습니다.

• 원인: 대시보드 상단에 신설된 [Season] 또는 [Segment / 차종] 필터가 활성화되어 있어, 해당 조건에 맞는 패턴들만 드롭다운에 노출되도록 동적 제한이 걸려 있기 때문입니다.
• 해결 조치 단계:
1. [제품 성능 비교] 화면 최상단의 [Season] 또는 [Segment / 차종] 필터 드롭다운이 특정 값(예: Summer 등)으로 선택되어 있는지 확인합니다.
2. 해당 필터들의 값을 전체 또는 공백으로 되돌리거나, 우측의 [초기화] 버튼을 클릭하여 필터 조건을 소거합니다.
3. 필터가 해제되면 9대 제조사 카드 내의 패턴 드롭다운 리스트에 전체 데이터베이스 패턴 목록이 다시 복원되어 표시됩니다.
