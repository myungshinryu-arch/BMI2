1. 통합 포털 홈 사용법

통합 포털 홈은 BM-Intelligence의 주요 분석 모듈로 이동하고, R&D 전략 가이드를 직접 관제하며, 통합 검색과 AI 기능을 실행하는 메인 화면입니다.

메인 화면 핵심 구성 및 사용 방법

1

통합 검색 기능 활용

• 설명: 화면 중앙의 검색창을 통해 타이어명, 제조사, 브랜드, 세그먼트, 컴파운드 원료, PLC 등 벤치마킹 관련 핵심 키워드를 기입해 탐색할 수 있습니다.
• 조작법:
1. 검색 입력창을 선택한 후 키워드(예: Michelin, Silica)를 입력합니다.
2. 입력 조건에 대응하는 타이어 BM 및 컴파운드 BM 데이터 검색 결과 목록이 드롭다운 형태로 하단에 실시간으로 표시됩니다.
3. 검색 목록에서 원하는 항목을 클릭하면 해당 분석 페이지의 상세 영역으로 즉시 연동되어 이동합니다.
4. 검색어가 명확하지 않은 경우에는 관련성이 높은 분석 모듈을 우선하여 추천해 줍니다.
5. 우측의 [X] 버튼을 누르면 입력한 텍스트가 모두 제거됩니다.

2

헤더 내 주요 통합 연동 버튼 활용

• 설명: 우측 상단에 배치된 연동용 바로가기 버튼들을 통해 신속하게 특정 핵심 기능들을 수행할 수 있습니다.
• 기능별 정의:
- [경쟁사 상품 Report 작성]: 현재 화면 데이터와 분석 context를 바탕으로 임원 보고용 경쟁사 상품 비교 리포트를 실시간으로 작성 및 인쇄하는 시스템 페이지로 이동합니다.
- [AI에게 질문하기]: 내부 데이터, BI News, Web Grounding을 유기적으로 교차 활용하는 Hybrid AI Agent와 실시간 질의응답을 수행할 수 있는 대화방으로 이동합니다.

3

핵심 분석 모듈 바로가기 링크 제어

• 설명: 통합 검색 패널 하단에 가로로 나열된 5대 핵심 분석 카드 영역입니다.
• 사용법: 시장/상품전략, Tire BM Report, Comp'd BM Report, 경쟁사 BI News, AI Simulator 등 카드 내 수록된 개별 메뉴 링크(예: [기술 경쟁력 Overview], [EV 친환경 특화 보드], [컴파운드 탐색기], [전체 뉴스 피드] 등)를 선택하면 해당 분석 서브 탭이 바로 활성화된 상태로 즉시 연결됩니다.

4

R&D 상품 전략 및 설비 투자 제안 콘솔 (R&D Strategic Console)

• 사용법:
1. 화면 하단의 필터 영역에서 **제조사** 및 **대상 제품군** 드롭다운을 변경하여 실시간 분석을 가동합니다.
2. 조건 변경 시 우측의 6대 성능 지표 비교 레이더 차트와 성능 상관관계 분석 수치가 실시간 렌더링됩니다.
3. 하단에 **성능 Gap Analysis**, **SKU 라인업 전략**, **글로벌 규제 대응 라벨링 목표**, **R&D 공정 설비 Capex 가이드라인**이 자동 수립되어 임원 및 의사결정권자에게 적합한 최적의 투자 방향을 제시합니다.
4. 헤더 최상단의 **[BM-Intelligence]** 브랜드 로고를 클릭하면 전체 동기화 연혁과 시스템 기획 로드맵이 수록된 프리미엄 **연혁(History) 팝업 모달**이 호출되며, 우측 상단 [✕] 아이콘 및 하단 닫기 단추로 종료할 수 있습니다.

Menu 02

2. 시장/상품전략 사용법

글로벌 Top4 제조사의 세그먼트별 매출, 판매량 정보 및 점유율 비중, 대표 타이어 세대별 성능 진화 궤적을 비교하여 중장기 로드맵을 확인하는 전략 가이드입니다.

내부 5대 핵심 분석 뷰(탭)별 정밀 화면 조작 단계

1

[기술 경쟁력 Overview] 다차원 필터링 및 성능 벤치마크 진단

• 실제 화면 위치: 시장전략 페이지의 첫 번째 메뉴 탭
• 주요 기능 및 조작법:
1. 화면 상단의 [대상 시장] 영역에서 North America 및 Europe 토글 버튼을 교차 클릭하여 타깃 권역을 변경합니다.
2. [대상 세그먼트], [대상 업체], [기준 분석 연도], [성능 데이터 소스] 드롭다운 셀렉트 박스를 조작하여 맞춤형 데이터를 구성합니다.
3. 필터 변경 시 상단의 4대 KPI 요약 카드(선택 조건 판매량, 벤치마크 1위 제품, 한국타이어 포지션, 최우선 R&D 보강 성능 항목)가 실시간으로 자동 갱신됩니다.
4. 중앙 좌측의 성능 6대축 레이더 차트에서 마우스 호버(Hover) 시 세부 점수가 툴팁으로 출력되며, 상단 범례(예: Michelin)를 클릭하면 해당 선을 숨기거나 켤 수 있습니다.
5. 우측의 YoY 성능 추이 차트 및 하단의 종합 성능 스코어카드(테이블)를 통해 세부 실측 수치와 연도별 성능 발달 궤적을 심층 판독합니다.

2

[대표 상품 세대별 성능] 브랜드 및 라인업 세대교체 추적

• 실제 화면 위치: 시장전략 페이지의 두 번째 메뉴 탭
• 주요 기능 및 조작법:
1. 필터바의 [분석 대상 제조사]에서 미쉐린, 콘티넨탈, 피렐리 버튼 중 분석할 제조사를 클릭합니다.
2. 제조사 선택 시 우측에 동적으로 생성되는 [세그먼트 / 라인업] 세부 제품군 버튼들을 클릭하여 조회 대상을 전환합니다.
3. 좌측 영역의 [세대별 공식 리플렛/브로셔 핵심 분석] 덱에서 각 세대(Gen 1 ➡️ Gen 2 ➡️ Gen 3)의 실물 리플렛 요약본과 핵심 기술 소구점을 수직 대조합니다.
4. 우측의 [세대별 세부 성능 항목 진화 트렌드] 차트 및 세대 셀렉터를 활용하여 성능 개선율과 제품 진화 흐름을 한눈에 포착합니다.

3

[대응 전략 & 로드맵] SWOT 분석 기반 미래 R&D 전략 및 Gap 진단

• 실제 화면 위치: 시장전략 페이지의 세 번째 메뉴 탭
• 주요 기능 및 조작법:
1. 화면 상단의 시장, 세그먼트, 기준 분석 연도 필터를 조작하여 자사에 미치는 최적의 미래 전략 시나리오를 구성합니다.
2. 선택된 시장 조건에 맞춰 지능형으로 렌더링되는 SWOT 분석 패널을 정독하여 강점(S), 약점(W), 기회(O), 위협(T) 인자가 도출합니다.
3. 하단 좌측의 [TO-BE 판매 & 기술력 확보 로드맵] 타임라인을 통해 고마진 UHP 인치업 믹스 강화, iON 라인업 글로벌 선점안 등의 세부 판매/기술 로드맵 과제를 수립합니다.
4. 하단 우측의 [상세 성능 격차 (Gap Analysis)] 패널에서 벤치마크 1위 제품 대비 당사 대표 모델의 성능 격차율(%) 프로그래스 바를 통해 정량적인 보강 수준을 판독합니다.

4

[Market & Sales Overview] 브랜드 다중 선택 및 거시 실적 추이 대조

• 실제 화면 위치: 시장전략 페이지의 네 번째 메뉴 탭
• 주요 기능 및 조작법:
1. [제조사 선택] 영역의 브랜드 칩(한국타이어, 미쉐린, 브리지스톤, 콘티넨탈, 굿이어, 피렐리, 금호타이어) 체크박스를 교차 클릭하여 비교군 브랜드를 다중 선택(Multi-select)합니다.
2. 실적 비교 차트 우측 상단의 메트릭 셀렉터에서 글로벌 매출액과 글로벌 판매량 버튼을 교차 클릭해 실적 지표를 동적으로 스위칭합니다.
3. 전사 실적 비교 차트 및 하단의 [글로벌 거시경제 동향 및 P-Q 성장 동학], [글로벌 베스트셀링 차종 및 OE 세그먼트 트렌드] 카드를 통해 글로벌 시장의 거시적 정황과 타이어 규격/하중 표준 상향 트렌드를 정독합니다.

5

[경쟁사 R&D 전략] Top 4 경쟁사 실시간 인텔리전스 및 비교 횡 매트릭스

• 실제 화면 위치: 시장전략 페이지의 다섯 번째 메뉴 탭
• 주요 기능 및 조작법:
1. 상단의 [글로벌 Top 4 경쟁사 실시간 리포트 조회] 바에서 미쉐린, 브리지스톤, 콘티넨탈, 굿이어 칩 버튼을 클릭합니다.
2. 칩 클릭 시 좌측의 R&D 테마 집중도 가중치 방사형 차트가 선택 브랜드의 지표로 실시간 드로잉되며, 우측의 다이내믹 요약 브리핑을 통해 핵심 기술 동향, 마케팅, 재무, 자사 영향도 인텔리전스 리포트를 확인합니다.
3. 하단의 [글로벌 4대 제조사 전략 비교 횡 매트릭스]를 활용하여 경쟁사들의 R&D 최우선 테마, 마케팅 소구, 가이던스 요약 정보와 [한국타이어 전용 Action Point]를 일목요연하게 횡형 비교 탐색합니다.

Menu 03

3. Tire BM Report 사용법

경쟁사 완성 타이어의 실측 성능 사양, PLC(제품수명주기) 타임라인 맵, EV 특화 친환경 타이어 필터 정보 및 사내 보고서 아카이브를 활용하는 조작법을 다룹니다.

4대 핵심 하위메뉴 및 실무 조작 절차

1

[종합 분석 대시보드] 탭 통계 지표 및 최근 리포트 조회

• 사용법:
1. 화면 상단의 **실시간 요약 통계 카드** 영역을 통해 총 벤치마킹 타이어 모델 수, EV 특화 모델 수, 완료된 EV 정밀 분석 수, 연계 Arena VPR 보고서 총합을 한눈에 관제합니다.
2. **카테고리 구성 비율** 리스트를 통해 세그먼트별(Summer, SUV, Winter-Alpin, All Weather) 비중을 실시간 파악합니다.
3. 하단 **최근 벤치마킹 리포트 핵심 리스트** 테이블에서 사내 승인이 완료된 최신 5대 보고서 목록의 부서, 기안자, 일자를 검토하고, [바로가기] 아이콘 단추를 클릭해 결재 완료 장표 원본으로 동적 연결합니다.

2

[PLC Timeline Map] 2차원 매트릭스 필터링 및 서랍(Drawer) 열기

• 사용법:
1. 화면 상단의 다단 서브 탭(Summer, Winter-Alpin, All Weather 등)을 선택해 수명 주기 시트를 활성화합니다.
2. 엑셀 스타일의 **세그먼트 및 제조사 멀티 선택 필터**를 조작하여 원하는 범위의 타이어만을 선택적으로 화면에 정밀 필터링합니다. [필터 초기화] 버튼으로 모든 조건을 원클릭 복구할 수 있습니다.
3. 가로 축(출시 연도: 2021년~2028년 이후)과 세로 축(세그먼트 및 브랜드 모델)으로 이루어진 다이어그램 타임라인에서 타이어 블록을 마우스 클릭합니다.
4. 화면 우측에서 **디테일 정보 서랍(Detail Drawer)**이 미려하게 호출됩니다. 서랍 내에서 **실물 트레드 패턴 이미지**, 제조사 뱃지, **제품 특이 기술 분석 소견**, 연계 승인 품의 보고서 상세 정보 및 [보고서 링크] 바로가기를 확인하고 검증합니다.

3

[EV 친환경 특화 보드] 칸반 카드 제어

• 사용법:
1. 친환경 전기자동차 전용 타이어들의 설계 벤치마킹 상태를 보여주는 전용 탭입니다.
2. 각 제조사별 EV 카드(예: Hankook iON, Michelin e.Primacy 등)의 특화 분석 사양을 칸반 열 형태로 추적합니다.
3. 개별 EV 타이어 카드를 좌클릭하면 마찬가지로 우측 **디테일 정보 서랍**이 슬라이딩 기동하여 전기차 전용 흡음재 장착 여부, 고하중 지수, 회전저항 등 특화 소견 및 연계 문서를 발췌 제공합니다.

4

[보고서 라이브러리] 81 EA Arena 연계 검색 및 PDF 뷰어

• 사용법:
1. 전체 81개에 달하는 벤치마킹 완료 기안 보고서 전수 아카이브 영역입니다.
2. **보고서 정밀 고속 필터링** 제어판을 활용하여 기안 부서, 기안자 드롭다운을 설정하고 제목/상품 키워드를 실시간 검색 기입해 대상을 필터링합니다.
3. 테이블의 보고서 제목 행을 클릭하거나 우측 [바로가기] 아이콘 단추를 누르면, Arena 전결 시스템에 등재된 PDF 장표 및 결재 원본 웹 문서로 안전하게 이동하여 열람이 가능합니다.

6. AI Simulator 사용법

벤치마크 타이어의 고무 원재료 슬라이더를 마우스로 직접 조절하여 가상 레시피를 가공 설계하고, AI가 예측 연산한 점탄성 차트 및 6대 성질 지수를 실시간 비교 추적하는 엔지니어링 패널입니다.

가상 배합 설계 및 시뮬레이션 상세 프로시저

1

벤치마크 타이어 선택

• 조작법: 화면 좌측 상단의 벤치마크 드롭다운 메뉴를 선택하여 대조 대상이 될 기준 경쟁사 모델(예: Michelin PS5 등)을 지정합니다. 지정 시 해당 타이어의 실제 계측 물리 정보가 초기 기준값으로 로드됩니다.

2

Virtual 레시피 디자인 (원료 함량 슬라이더 튜닝)

• 조작법:
1. 좌측 배합 제어부의 **Silica (phr)**, **TESPT (%)**, **Oil (phr)** 등 함량 조절 슬라이더 노브를 좌우로 드래그하여 가상의 신규 배합안(Virtual Recipe)을 실시간으로 설계합니다.
2. 더 정밀하게 소수점 단위 미세 조정을 수행하고 싶을 때에는, 슬라이더 노브를 1회 마우스 클릭하여 선택한 후 키보드 좌우 방향키(←, ➡️)를 누르면 0.1 phr / 0.1% 간격으로 수치 미세 튜닝이 구동됩니다.

3

ARES tanδ Temperature Sweep 분석

• 사용법: 배합 슬라이더의 변경 위치에 연동되어 우측 차트 내 가상 예측선(점선 형태)이 실시간으로 갱신 렌더링됩니다. 온도별 tanδ 곡선의 변동 거동(예: 0℃ 회전저항 특성, 60℃ 연비/제동 밸런스 등)을 눈으로 면밀하게 추적 비교할 수 있습니다.

4

성능 비교 분석 보드 (정량 지표 점수 대조)

• 사용법:
1. 화면 우측 영역의 6대 성능 밸런스 레이더 차트와 상세 스코어 보드를 확인합니다.
2. 사용자가 설계한 Virtual 배합과 실제 Benchmark 타이어 간의 **Wet 제동 성능, 연비(RR), 내마모 점수** 등 주요 6성 지수 격차를 100점 만점 평점으로 대조 관제하여 배합 최적 지점을 도출해 냅니다.

5

AI 레시피 개선 가이드 실행

• 사용법: 가상 배합 튜닝 도중 성능을 더욱 비약적으로 극복하기 위해, 제어판 하단의 **[AI 레시피 개선 가이드]** 버튼을 선택합니다. 인공지능이 내장 데이터 기반으로 현 슬라이더 조건의 임계점을 분석하여 우수한 목표 성능 평점에 도달할 수 있는 추천 개선 후보 배합안 리스트를 연출해 줍니다.

6

추천 배합 적용 및 반영

• 사용법: 개선 가이드 팝업창에서 추천된 AI 후보 배합 조건들 중 적합한 설계안을 선택하고 **[배합 적용]** 버튼을 클릭합니다. 즉시 해당 가이드 배합 수치가 좌측 가상 슬라이더 노브 위치로 정합 자동 세팅되며 차트 분석값이 즉각 일체 재반영됩니다.

2

지능형 리포트 자동 생성 단추 클릭

3. 미리보기에서 다크 테마 인포그래픽 정렬 정합성을 최종 확인하고 인쇄 또는 PDF 저장 버튼을 선택합니다.

8. AI에게 질문하기 사용법

내부 지식 데이터베이스(RAG), 실시간 수집된 최신 경쟁사 기술 뉴스, Web Grounding 기술을 유기적으로 하이브리드 교차 구동하여 정교한 자연어 답변을 반환하는 AI Insight Agent 대화방입니다.

Hybrid AI Agent 전송 및 식별 뱃지 판독법

1

질문 유형에 따른 지능형 하이브리드 라우팅 자동 연동

• 조작법: 챗방 하단 텍스트 창에 타이어 성능, 컴파운드 설계, 경쟁사 최근 출시 일정, R&D 트렌드 등에 관한 궁금한 사항을 자유롭게 자연어로 입력하고 전송 버튼을 누릅니다.
• 라우팅 작동 구조:
- **내부 데이터 질문**: 대시보드 물성 정보나 내부 저장 자료(RAG context)를 최우선 매핑하여 팩트 중심의 정량 수치로 정밀하게 답변합니다.
- **최신 출시/뉴스/동향 질문**: 내부 데이터에 수록되지 않은 최신성/외부 시장 정보는 실시간 구글 웹 검색을 가동해 Web Grounding 기반의 가장 공신력 있는 내용으로 발췌하여 기술해 줍니다.
- **하이브리드 결합 질문**: 자사 수치와 외부 최신 동향을 통합 대조하는 질문의 경우, 내부 수치 데이터셋과 실시간 외부 웹 기사 정보를 일련적으로 조합해 하이브리드 분석으로 명쾌히 풀어내 줍니다.
- **시뮬레이터 관련 질문**: 가상 배합 설계나 tanδ 분석 시, 현재 튜닝 중인 내부 Simulator context 값을 우선 활용하여 대답을 완성합니다.

답변 근거 구분용 5대 시스템 식별 뱃지 확인 요령

• 설명: 대화형 버블 상단 또는 우측 영역에 출력되는 컬러풀한 시스템 전용 뱃지를 활용해 인공지능이 생성한 답변의 원천 근거를 한눈에 신뢰 및 검증할 수 있습니다.

Gemini LLM : Google Vertex AI의 최상위 Gemini LLM 추론 모델을 사용했음을 인증하는 표식입니다.
내부 데이터 기반 : 답변이 외부 추정이 아니라 엄선된 사내 벤치마킹 데이터베이스(RAG) 원천 수치를 참조해 구성되었음을 보증합니다.
Web Grounding 사용 : 실시간 웹서치 기술(Google Search Grounding)과 교차 검증을 수반하여 도출된 신뢰할 만한 외부 팩트 정보임을 뜻합니다.
Hybrid 분석 : 내부 정밀 물성 수치와 외부 시장 트레이드 동향 기사를 하나의 논리로 동적 결합했음을 표기합니다.
Local Fallback : 네트워크 정체나 API 에러 유발 시 오답을 임의 추론(Hallucination)하는 대신, 로컬에 저장 보관된 기본 지식 시나리오로 대체하여 정답을 안전하게 대체 표출한 상태를 나타냅니다.

9. FAQ 및 오류 대처

플랫폼 주요 모듈을 이용하시다 예기치 못한 작동 정지나 데이터 누락 현상 마주 시 현업 연구원이 신속히 조치할 수 있는 트러블슈팅 매뉴얼입니다.

Q1

Q. 분석 화면이나 대시보드 내 특정 영역에 그래프나 정량 수치 표 데이터가 전혀 노출되지 않습니다.

• 원인: 데이터베이스 API 커넥터 모듈의 순시적 응답 지연 또는 로컬 웹브라우저 리소스 로딩 캐시 꼬임이 주 원인입니다.
• 해결 조치 단계:
1. 사용 중인 브라우저 화면에서 **[Ctrl + F5]** 키를 동시에 입력하여 강력하게 강제 캐시 초기화 및 새로고침을 실행해 줍니다.
2. 새로고침 후에도 문제가 지속되면, 로컬 네트워크의 연결 상태와 시스템 백엔드 API 서버 포트 통신망이 가동 중인지 IT 부서나 인프라 관리자에게 연결 상태 점검을 조율하십시오.

Q2

Q. AI에게 질문하기 및 경쟁사 상품 Report 생성 도중 답변 근거 뱃지에 'Local Fallback' 상태만 표시됩니다.

• 원인: 백엔드 uvicorn API 서비스 장애, Vertex AI Cloud 인증 계정 토큰 만료, 또는 허용 IP 이탈 등이 원인입니다.
• 해결 조치 단계:
1. FastAPI 서버 엔진(`app:app`)이 백그라운드에서 정상 가동되고 있는지 CLI 명령창 상태를 진입 조회합니다.
2. Cloud Run API 주소 상태 및 Vertex AI 프로젝트(`hk-hackathon-01`), 리전(`global`)에 대한 로컬 ADC(Application Default Credentials) 권한 자격 증명 활성화 상태를 면밀히 점검하십시오.

Q3

Q. 메인 통합 검색창에 검색어를 기입했는데 결과 없음 상태만 표출되어 이동이 불가능합니다.

• 원인: 오탈자가 있거나 너무 축약된 기호를 입력했을 경우 시스템 매핑 범위를 초과하게 됩니다.
• 해결 조치 단계:
1. 입력어를 조금 더 대중적인 제조사명(예: Michelin, Kumho), 모델명(예: PS5, Ventus), 세그먼트명, 혹은 특정 고무 원재료 키워드로 풀네임을 더 구체적으로 정밀하게 기입해 보시면 정상 추천 필터가 즉각 되살아납니다.

Q4

Q. AI Simulator에서 원재료 함량 슬라이더 바를 분명 이동했는데도 오른쪽 tanδ 예측 그래프가 변동하지 않고 정지해 있습니다.

• 원인: 벤치마크 대상 타이어 모델이 정상적으로 체크 지정되지 않았거나, 브라우저 스크립트 실행 스레드의 일시 지연 상태입니다.
• 해결 조치 단계:
1. 좌측 최상단의 대조 벤치마크 타이어 드롭다운 선택 메뉴에서 정확한 비교 모델이 유효하게 로딩 지정되어 있는지 다시 한 번 명확하게 체크 기입하십시오.
2. 슬라이더 노브를 살짝 클릭하여 마우스 포커스를 준 뒤, 키보드 방향키를 톡톡 쳐서 입력 상태 이벤트를 강제로 재호출하시면 그래프 선이 실시간으로 부드럽게 다시 동기화 갱신됩니다.
